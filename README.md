# LadyBird Accounting Solutions Website

A professional static website for Samantha Taylor, sole trader and registered BAS Agent trading as LadyBird Accounting Solutions, providing comprehensive accounting and payroll services to small businesses across Australia.

## Features

- **Responsive Design**: Fully responsive website that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Single Page Application**: All content organized in one scrollable page with smooth navigation
- **Professional Branding**: Color scheme based on the LadyBird logo (reds and dark greys)

## Website Sections

1. **Navigation**: Sticky header with smooth scroll links
2. **Hero Section**: Eye-catching introduction with mission statement
3. **Services**: Comprehensive list of services including BAS, bookkeeping, payroll, GST, and more
4. **About**: Detailed background about Samantha Taylor and LadyBird Accounting Solutions
5. **Contact**: Contact information and call-to-action
6. **Footer**: Professional footer with branding

## How to Use

1. **Open Locally**: Simply open `index.html` in any modern web browser
2. **Deploy to Web**: Upload all files to any web hosting service maintaining the directory structure

## Deployment Options

### Option 1: GitHub Pages
1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select the main branch as source
4. Your site will be available at `https://yourusername.github.io/ladybirdWebsite`

### Option 2: Netlify
1. Drag and drop the entire folder to [Netlify](https://app.netlify.com/)
2. Your site will be live instantly with a custom URL

### Option 3: Traditional Web Hosting
1. Upload all files via FTP to your web hosting service
2. Ensure the directory structure is maintained

## Customization

### Updating Content
- Edit `index.html` to modify text content
- Update contact information in the Contact section
- Add or remove services as needed

### Changing Colors
- All colors are defined in CSS variables in `css/styles.css` (lines 18-26)
- Modify these variables to change the entire color scheme:
  - `--primary-color`: Main brand color (red)
  - `--secondary-color`: Dark accent color
  - `--accent-color`: Lighter red for hover states

### Adding Images
- Place new images in the `images/` folder
- Reference them in HTML using relative paths: `images/your-image.png`
